Place compiled native protection binaries here.

Expected paths:
- /native/win64/zovguard.dll
- /native/win32/zovguard.dll
- /native/linux64/libzovguard.so
- /native/linux32/libzovguard.so
- /native/macos/libzovguard.dylib

If no native binary is present, the client can still start in non-strict mode.
